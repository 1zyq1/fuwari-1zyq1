# 欢迎来到我的博客！

这是基于 [Fuwari](https://github.com/saicaca/fuwari) 模板搭建的博客，分享技术心得、学习笔记和生活点滴的空间。

**技术栈：**

*   **构建工具**: [Astro](https://astro.build)
*   **样式**: [Tailwind CSS](https://tailwindcss.com)

 **功能特性**

*   基于 Astro 和 Tailwind CSS 开发
*   流畅的动画和页面过渡
*   亮色 / 暗色模式
*   自定义主题色和横幅图片
*   壁纸 ui 美化
*   响应式设计
*   评论
*   搜索
*   文内目录
*   数据统计

希望我的博客能为你带来一些启发和帮助。欢迎随时浏览和交流！
